# Emotion Detector App

## Description
This project is an AI-based web application that detects emotions from text using IBM Watson NLP.

## Features
- Emotion detection (anger, joy, sadness, etc.)
- Flask web interface
- Error handling
- Unit tested

## Installation
git clone <your_repo>
cd Emotion-Detector
pip install -r requirements.txt

## Usage
python3 server.py

## Example
Input: "I am very happy"
Output: joy
